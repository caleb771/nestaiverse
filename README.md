# NestAI Prototype

Early prototype of NestAI — an AI-powered interior design system that
understands rooms as structured environments instead of generating images.

## Concept

NestAI operates on a **world state**:

Goal → Reasoning → Spatial Decisions → Blender Visualization

Blender acts as a renderer, while design logic exists independently.

## Features

- Semantic furniture catalog
- Constraint-based placement
- Collision avoidance
- Sketchfab asset integration
- Procedural room generation

## Setup

1. Install Blender
2. Clone repo
3. Place GLB assets in `/assets`
